# DEDISP: C++ Reference

This directory contains the `dedisp` (C++) reference code, based on the Fourier Domain Dedispersion method developed by [C.G. Bassa et al. (2021)](https://git.astron.nl/RD/dedisp/).

## Formatting

This project is formatted using `clang-format`:
```bash
clang-format -i $(find src/ include/ -name "*.[hc]pp")
```
